## 🛠️ Tech Stack

Our website is powered by a modern, high-performance tech ecosystem:

- **React 18** - For a dynamic, high-fidelity UI.
- **Vite** - Lightning-fast development and build pipeline.
- **Tailwind CSS** - For sleek, dark-mode-first styling with precision.
- **Framer Motion** - Delivering fluid, premium animations.
- **Lucide React** - A curated suite of beautiful, modern icons.

## 📂 Project Structure

```text
curosweb/
├── src/
│   ├── components/    # Reusable UI components (Navbar, Footer, etc.)
│   ├── pages/         # High-impact page views (Home, About, Portfolio)
│   ├── assets/        # Visual assets and design tokens
│   └── App.jsx        # Routing and core logic
├── public/            # Static assets
└── tailwind.config.js # Custom design system tokens
```

## ⚡ Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Abhishek30kush/curosweb.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Ignite the development server:**
   ```bash
   npm run dev
   ```
